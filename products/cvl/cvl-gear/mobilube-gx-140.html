美孚路宝™GX 140 (Mobilube GX 140)

美孚路宝 GX 140 是一款高性能的车辆齿轮润滑油，采用优质基础油和先进的添加剂调配而成。该产品专为中型和重型商用车的驱动桥和手动变速箱设计，符合 API GL-5 质量级别要求，提供卓越的极压抗磨保护和持久的润滑性能。

产品简介

美孚路宝 GX 140 齿轮油的突出优势:

  • ✅ 卓越的极压抗磨性: 在重载工况下为齿轮和轴承提供出色保护，减少磨损
  • ✅ 出色的氧化稳定性: 高温条件下保持油品性能稳定，延长换油周期
  • ✅ 优异的水分离性能: 快速分离水分，防止乳化，保护传动系统免受腐蚀
  • ✅ 良好的热稳定性: 有效控制油泥和沉积物生成，保持系统清洁
  • ✅ 优异的密封件适应性: 与常见密封材料兼容，防止泄漏
  • ✅ 长效润滑保护: 即使在严苛工况下也能提供持久的润滑性能

典型应用

  • 🚛 中型和重型卡车驱动桥
  • 🚌 客车和巴士的手动变速箱
  • 🏗️ 工程机械的传动系统
  • 🔧 其他需要 API GL-5 及 SAE 140 润滑油的车辆齿轮装置
