美孚黑霸王™发动机机油 (普黑) 系列 (Mobil Delvac™ Regular Series)

美孚黑霸王™发动机机油 (普黑) 系列是经典的柴油发动机油产品，专为满足国二/国三排放标准的商用车柴油发动机设计。该系列包括 CF-4(红盖)、超级 (银盖) 和经典 (黑盖) 三款产品，全面覆盖从经济实用到高性能的不同需求。所有产品均符合 API CF-4标准，采用优质基础油和平衡的添加剂包调配而成，具备出色的清洁分散性、抗磨损保护和抗氧化稳定性，有效延长发动机使用寿命，降低维护成本，是城市配送车辆、轻型卡车、农用运输车及一般工况柴油发动机的理想选择。

美孚黑霸王™CF-4 发动机机油 (红盖)

API CF-4 级别，传统技术配方，自然吸气柴油发动机经济实用之选。

应用范围：适用于国二/国三排放标准的自然吸气柴油发动机，城市配送车辆、轻型卡车及一般工况车辆

美孚黑霸王™超级发动机机油 (银盖)

API CI-4级别，合成科技配方，高性能涡轮增压柴油发动机理想选择。

应用范围：适用于国四/国五排放标准的涡轮增压柴油发动机，中型卡车、工程机械及高负荷运输车辆

美孚黑霸王™CH-4 发动机机油 (黑盖)

API CH-4级别，基础柴油机油，日常保养经济之选。

应用范围：适用于国二/国三排放标准的轻载柴油发动机，城市配送、短途运输及一般工况运营车辆
